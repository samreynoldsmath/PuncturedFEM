# Commit checklist

## Before commit
- [ ] update `TODO.md`
- [ ] update `CHANGELOG.md`
- [ ] run `isort`
  - [ ] source code
- [ ] run `black` / `flake8` / `mypy`
  - [ ] source code
  - [ ] tests
  - [ ] examples
- [ ] all tests pass
- [ ] convert Juptyer notebooks to Python scripts
- [ ] all examples work

## After commit
- [ ] update tag
- [ ] push to GitHub
